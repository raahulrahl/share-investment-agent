# Technical Analysis Specialist

## Overview
The Technical Analysis Specialist specializes in analyzing price movements, trading patterns, and market indicators to generate short-term trading signals. This specialist uses various technical indicators and chart patterns to identify potential trading opportunities and market trends.

## Core Responsibilities

### Indicator Analysis
- **Trend Indicators**: Moving averages, MACD, ADX
- **Momentum Indicators**: RSI, Stochastic, Williams %R
- **Volatility Indicators**: Bollinger Bands, ATR, VIX
- **Volume Indicators**: OBV, Volume Profile, Money Flow Index

### Pattern Recognition
- **Chart Patterns**: Head and shoulders, triangles, flags, pennants
- **Candlestick Patterns**: Doji, hammer, engulfing patterns
- **Support/Resistance Levels**: Key price levels, trend lines
- **Reversal Patterns**: Double tops/bottoms, triple patterns

### Signal Generation
- **Buy Signals**: Oversold conditions, bullish patterns, breakouts
- **Sell Signals**: Overbought conditions, bearish patterns, breakdowns
- **Hold Signals**: Neutral conditions, consolidation phases
- **Confidence Scoring**: Probability assessment for each signal

## Technical Indicators

### Trend Analysis
```python
# Moving Averages
SMA_20 = SimpleMovingAverage(close_prices, period=20)
SMA_50 = SimpleMovingAverage(close_prices, period=50)
EMA_12 = ExponentialMovingAverage(close_prices, period=12)

# MACD
MACD_line = EMA_12 - EMA_26
Signal_line = EMA(MACD_line, period=9)
Histogram = MACD_line - Signal_line
```

### Momentum Analysis
```python
# RSI (Relative Strength Index)
RSI = 100 - (100 / (1 + RS))
where RS = Average_Gain / Average_Loss

# Stochastic Oscillator
%K = ((Current_Close - Lowest_Low) / (Highest_High - Lowest_Low)) * 100
%D = SMA(%K, period=3)
```

### Volatility Analysis
```python
# Bollinger Bands
Middle_Band = SMA(close_prices, period=20)
Upper_Band = Middle_Band + (2 * Standard_Deviation)
Lower_Band = Middle_Band - (2 * Standard_Deviation)

# ATR (Average True Range)
TR = max(High-Low, abs(High-Previous_Close), abs(Low-Previous_Close))
ATR = SMA(TR, period=14)
```

## Analysis Framework

### 1. Trend Identification
- **Uptrend**: Higher highs and higher lows
- **Downtrend**: Lower highs and lower lows
- **Sideways**: Horizontal price movement
- **Trend Strength**: Measured by ADX indicator

### 2. Momentum Assessment
- **Overbought**: RSI > 70, Stochastic > 80
- **Oversold**: RSI < 30, Stochastic < 20
- **Momentum Divergence**: Price vs indicator divergence
- **Momentum Confirmation**: Multiple indicators aligning

### 3. Volume Analysis
- **Volume Confirmation**: High volume supports price moves
- **Volume Divergence**: Price moves without volume support
- **Volume Patterns**: Accumulation, distribution, climax
- **Volume Trends**: Increasing/decreasing volume patterns

### Confidence Scoring
- **High Confidence**: 3+ indicators aligning
- **Medium Confidence**: 2 indicators aligning
- **Low Confidence**: Single indicator signal
- **No Signal**: Contradictory indicators
